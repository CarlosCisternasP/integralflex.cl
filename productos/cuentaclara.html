FUNCIONALIDADES

Qué hace CuentaClara

📸 OCR de boletas y facturas

Sube PDF o foto y el sistema extrae monto, fecha, proveedor, folio, subtotal e IVA.

✏️ Revisión editable

El colaborador confirma o corrige los datos antes de enviar la rendición.

📱 App Android

APK propia además de la web móvil con menú optimizado para teléfono.

💬 Asistente WhatsApp

Wizard de carga de gastos vía WhatsApp.

📊 Dashboard ordenado

Estado de rendiciones por colaborador, centro de costo y período.

✅ Flujo de aprobación

Rendiciones asociadas al usuario con revisión y aprobación.

👥 Multiusuario con roles

Colaboradores, aprobadores y administración.

🗄️ PostgreSQL + Docker

Backend robusto y despliegue con Docker Compose.
